# TryHackMe Write-ups

Welcome to my TryHackMe portfolio.

This repository documents my hands-on learning journey through TryHackMe. Each write-up highlights the methodology, tools, and lessons learned while solving cybersecurity labs.

## Objectives

- Develop practical penetration testing skills
- Improve enumeration and vulnerability assessment techniques
- Document professional methodologies
- Strengthen reporting skills
- Build a cybersecurity portfolio

## Repository Structure

Each room will include:
- Room Overview
- Objectives
- Tools Used
- Methodology
- Key Findings
- Lessons Learned

> Note: I respect TryHackMe's content guidelines. These write-ups focus on learning outcomes and methodology rather than publishing walkthroughs for active rooms.
